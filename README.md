# Futebol API ⚽
API desenvolvida em .NET 8 para gestão de elencos de times de futebol.

### Tecnologias:
- C# / .NET 10.0 (LTS)
- Entity Framework Core 10.0
- PostgreSQL (Banco de dados)
- Docker Desktop
- Swagger / OpenAPI
- API-Football (Integração externa)

### Como rodar:
1. `dotnet restore`
2. `dotnet ef database update`
3. `dotnet run`
